API

GET: https://unhashit.com/HashReversing/JSON/{algorithm}/digest?value={value}
Description

Returns hash value calculated with specified algorithm

URL

https://unhashit.com/HashReversing/JSON/{algoithm}/digest

Parameters
  • value - value to be processed with hash function
Supported algorithms
  • {{item}}
Example curl request

curl "https://unhashit.com/HashReversing/JSON/MD5/digest?value=HelloWorld"

Example response

{
"digest":"68e109f0f40ca72a15e05cc22786f8e6",
"value":"HelloWorld",
"algorithm":"MD5"
}

GET: https://unhashit.com/HashReversing/JSON/{algorithm}/value?digest={value}
Description

Returns a value which if passed to hash function will return specified hash value or NULL if value was not found

URL

https://unhashit.com/HashReversing/JSON/{algoithm}/value

Parameters
  • digest - hash value to find
Supported algorithms
  • {{item}}
Example curl request

curl "https://unhashit.com/HashReversing/JSON/MD5/value?digest=68e109f0f40ca72a15e05cc22786f8e6"

Example response

{
"digest":"68e109f0f40ca72a15e05cc22786f8e6",
"value":"HelloWorld",
"algorithm":"MD5"
}
