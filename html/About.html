What is Unhashit.com?

With Unhashit.com you can decrypt hashes into original text. You also can generate hashes using different algorithms. Hashes are used for many applications like security, authentication and others.

What is a Hash?

A hash, or hash function is a function that can be used to convert data of any size to a much smaller size. It's like a fingerprint of the data. The result of a hash function are called hash code, digest, hash value or simply hash. If the data changes just a little bit, the resulting hash will change completely. Until now it was very difficult to reconstruct data just from the hash value. Unhashit.com has built a giant database of over 1.5 billion precomputed inputs and their corresponding hashes. With that database you can reverse / decrypt / reconstruct a hash into it's initial form super fast.

What hashes does Unhashit.com support?

We support MD5, SHA1, SHA256 and SHA512. Unhashit.com is adding new hashes constantly.

Where do the hashes come from?

Every time a user of Unhashit.com generates a hash, the hash is also stored for later reverse/decryption.
